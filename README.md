# Geometric Progression — приклад (C#)

Невеликий консольний проєкт, який містить клас `GeometricProgression`.

Опис:

- Клас характеризується першим членом `a1`, знаменником `q` та кількістю членів `n`.
- Метод `IsLastLargest()` визначає, чи є останній член прогресії найбільшим серед усіх членів.

Як збілдити та запустити (Windows PowerShell):

```powershell
cd d:\git-repo\nuwm_repo_oop
dotnet build
dotnet run --project .
```

Примітка: потрібен встановлений .NET SDK (підтримує target framework net7.0 або змініть у файлі .csproj).
